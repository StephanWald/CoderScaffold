Create `scripts/push-templates.sh`: a non-interactive bash wrapper around the `coder`
CLI that (1) ensures the operator is authenticated to the Coder server, then (2) pushes
EVERY workspace template under `templates/` in one run.

## coder CLI interface (use exactly these — CLI not present to --help here)

- `coder whoami` — exits non-zero when not authenticated (session detection)
- `coder login <url> [--token <token>]` — authenticate; url and token both optional
- `coder templates push <name> --directory <dir> --yes` — push one template, `--yes`
  skips the interactive confirmation so the loop is non-interactive

Keep flags conservative and documented; do not invent flags beyond these.
